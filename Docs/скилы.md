# Скилы

Скил — активируемая способность. Содержит активатор и набор проков. На каждый скил навешивается один модификатор (см. [модификаторы.md](модификаторы.md)).

Игрок имеет 4 активных слота. Скилы находятся по ходу игры: в сундуках, выпадают с врагов, даются за завершение локации.

# Правила документации AlchJam

## Язык

Все файлы диздока пишутся на **русском языке**.

## Именование файлов

- Формат: `название.md`
- Название — на русском, строчными буквами, слова разделяются `_`

## Структура документации

Один файл = одна тема. Не совмещать несколько несвязанных тем в одном файле.

## Основные файлы

| Файл | Содержание |
|------|------------|
| [концепт.md](концепт.md) | Концепт игры, механики, цикл геймплея, технические рамки |
| [скилы.md](скилы.md) | Таблица скилов с типами, описаниями и требованиями |
| [модификаторы.md](модификаторы.md) | Таблица модификаторов с категориями и совместимостью |
| [глоссарий.md](глоссарий.md) | Глоссарий терминов проекта |

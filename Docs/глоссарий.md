# Глоссарий

# Большая особь
Главный слизень, который увеличивается, чем больше в нем находятся малых слизней

# Малая особь
Одна единица слизня вне большой особи

# Скил
Активируемая способность. Содержит активаторы

# Активатор
Условие срабатывания эффектов скила. Например: попадание проджектайлом, вхождение в область, активация скила.

# Модификатор
Дополнение к скилу, навешиваемое отдельно. Меняет урон, КД, длительность или тип эффекта.

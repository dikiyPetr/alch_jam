# AlchJam — Концепт игры

## Общее описание

Упрощённый Diablo-like экшен

Игрок управляет персонажем-слизнем, чья ключевая особенность — способность делиться на несколько особей меньшего размера и управлять ими как единым персонажем.
Это основа всей игровой механики.

---

## Основная механика: деление и слияние

Персонаж может делиться на маленьких слизней или сливаться в один большой. Это происходит при использовании скилов

---

## Цель
Игроку нужно пройти до цели прокачиваясь по пути. Путь представляет аналог карты из slay the spire

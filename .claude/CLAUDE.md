# Правила проекта AlchJam

## Unity — создание объектов и компонентов

- **Запрещено** создавать компоненты программно через `gameObject.AddComponent<T>()`.
- **Запрещено** создавать GameObject-ы через `new GameObject(...)` в рантайме.
- Все объекты сцены создаются через **префабы**, ссылки на которые хранятся в **публичных или [SerializeField] полях** MonoBehaviour.
- Изменения сцены (добавление объектов, настройка компонентов) делаются через **редактор Unity** или MCP-инструменты сцены.

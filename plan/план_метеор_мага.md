# План: метеор мага

## Текущее состояние

`ProjectileAreaDamage` — летит горизонтально к цели, при прибытии наносит урон через `AreaDamageMono`. Никакого предупреждения нет.

## Цель

1. Показать предупреждение (decal-круг) на земле под целью
2. Через задержку: заспавнить метеор высоко над зоной, он падает вниз
3. При ударе: убрать предупреждение, активировать зону урона на N секунд

---

## Фазы атаки

```
[Маг кастует]
      ↓
[Фаза Warning]  — decal-круг на земле, X сек
      ↓
[Фаза Fall]     — метеор падает сверху, видно визуально
      ↓
[Фаза Impact]   — зона урона активна N сек, decal гаснет
      ↓
[Destroy]
```

---

## Что нужно реализовать

### Новые компоненты

| Компонент | Описание |
|---|---|
| `MeteorAttack` | Главный MonoBehaviour: управляет фазами через coroutine или state. Запускается через `Launch(Vector3 targetPos)` |
| `MeteorWarningDecal` | Дочерний объект: визуальный круг на земле (через Projector или Decal), масштаб = радиус зоны |
| `MeteorFallVisual` | Дочерний объект: спрайт/меш метеора, стартует высоко над целью, падает вниз |

### Переиспользуем

- `AreaDamageMono` — зона урона при импакте, уже готова, просто активируется в нужный момент
- `event OnDamageDealt` — оставить на `MeteorAttack` для статистики мага
- `DamageRangeFinder` — вешается на объект зоны урона как обычно

---

## Структура префаба

```
MeteorAttack (MeteorAttack.cs, DamageSourceMono)
├── Warning (MeteorWarningDecal.cs)        ← decal на земле
├── MeteorVisual (MeteorFallVisual.cs)     ← спрайт метеора
└── DamageZone (AreaDamageMono + DamageRangeFinder + Collider)
```

---

## Логика `MeteorAttack`

```
Launch(targetPos):
  1. Поставить Warning на targetPos — показать decal
  2. Ждать _warningDuration сек
  3. Спавнить MeteorVisual над точкой (targetPos + Vector3.up * _spawnHeight)
  4. Анимировать падение MeteorVisual → targetPos за _fallDuration сек
  5. При прибытии: скрыть Warning + Visual, активировать DamageZone
  6. Ждать _impactDuration сек
  7. Destroy(gameObject)
```

### Поля `MeteorAttack`

```
[SerializeField] float _warningDuration = 1.5f   // время подсветки до удара
[SerializeField] float _spawnHeight = 10f         // высота спавна метеора
[SerializeField] float _fallDuration = 0.4f       // время падения
[SerializeField] float _impactDuration = 2f       // время активной зоны урона
```

## Изменения в `MageDummy`

- Заменить `ProjectileAreaDamage _projectilePrefab` → `MeteorAttack _meteorPrefab`
- `Attack()`: `Instantiate(_meteorPrefab)` → `meteor.Launch(targetPos)`
- Подписка на `meteor.OnDamageDealt` — без изменений

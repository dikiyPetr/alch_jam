# План: создание врагов

## Текущее состояние

### Реализовано
Прототип врага, что следует за игроком 

### Не реализовано
- Смерть врага — `IHittable` без реакции на 0 HP
- Параметры врага — нет ScriptableObject/компонента с HP, уроном, скоростью
- Настроенные префабы 
- Несколько типов врагов

---

## Цель

Три готовых к расстановке префаба врагов, каждый с настроенными параметрами, работающей атакой и смертью.
Бос, возможно
---

## Приоритеты

### P0 — минимально рабочий враг
1. **Смерть врага** — добавить HP-компонент (или через `IHittable`), при 0 HP уничтожить GameObject, убрать из пула `Group`.
2. **AttackState** — враг останавливается в радиусе атаки, наносит урон слайму с кулдауном, возвращается в `FollowState` если цель вышла из радиуса.
3. **Префаб "Рядовой"** — настроить `DummyAi` + `DummyUnit` с реальными значениями: HP 100, урон 10, скорость 3, радиус атаки 1.5.
